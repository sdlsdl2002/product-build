# Project Blueprint

## Overview

This project is a lottery number generator. It provides users with a simple and visually appealing interface to generate random lottery numbers.

## Feature Set

*   **Lottery Number Generation:** Generates 6 unique random numbers from 1 to 45.
*   **User-Friendly Interface:** A clean and simple interface for generating numbers.
*   **Modern Aesthetics:** The UI will feature a clean layout, good typography, and a pleasant color scheme.

## Implementation Plan

### 1. **HTML Structure (`index.html`)**
    *   Set up the basic HTML5 document structure.
    *   Create a main container for the application.
    *   Add a title for the page.
    *   Create a container to display the generated lottery numbers.
    *   Add a button to trigger the number generation.
    *   Link to the CSS and JavaScript files.

### 2. **Styling (`style.css`)**
    *   Apply a clean and modern design to the page.
    *   Style the number display to be clear and readable.
    *   Style the "Generate" button to be intuitive and attractive.
    *   Ensure the layout is responsive and looks good on different screen sizes.

### 3. **JavaScript Logic (`main.js`)**
    *   Implement a function to generate 6 unique random numbers between 1 and 45.
    *   Add an event listener to the "Generate" button.
    *   When the button is clicked, call the number generation function and display the results in the designated container.
